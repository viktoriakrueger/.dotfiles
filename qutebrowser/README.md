## Installation

### osx:

`brew install qutebrowser`

- copy userscripts

`cp -r ~/.dotfiles/qutebrowser/.qutebrowser/userscripts ~/Library/Application\ Support/qutebrowser`

### arch linux:

`sudo pacman -S qutebrowser`
